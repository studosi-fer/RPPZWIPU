Lista popularnosti programskih jezika (TIOBE indeks prosinac 2019):
Java
general-purpose programming language that is class-based, object-oriented, and designed to have as few implementation dependencies as possible.
C
computer programming book written by Brian Kernighan and Dennis Ritchie, the latter of whom originally designed and implemented the language, as well as co-designed the Unix operating system with which development of the language was closely intertwined.
Python
interpreted, high-level, general-purpose programming language.
C++
general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language, or "C with Classes".
